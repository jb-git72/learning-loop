# Birchal CSF campaign client directory

Marketing assets for Birchal's own CSF raise on the Birchal platform.

## Layout

- `loop/birchal-ad-variants/` -- Meta ad JSONs (one per variant)
- `loop/birchal-ad-variants/summary.json` -- variant metadata index

## Compliance posture

All copy must follow:
- `compliance-au` skill, CSF subsection (Part 6D.3A, INFO 217, RG 261/262)
- `direct-response-copy` skill, `references/regulated-industries-au.md`
- Verbatim CSF risk warning at the foot of primary text
- No preferential-terms language (s 738G equal terms)
- No offer-like verbs pre-offer (no 'invest now', 'buy shares')
